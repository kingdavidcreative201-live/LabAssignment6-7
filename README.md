# CPSC6548 Lab 6-7: Inventory Management System

## Assignment Requirements
- 4 parallel arrays with 20 items each
- String[] productName, double[] productPrice, int[] productQuantity, char[] productCategory
- Multiple reporting functions with formatted output
- Bubble sort with index sorting algorithm
- Menu-driven interface

## Features Implemented
| Function | Description |
|----------|-------------|
| displayReport() | Full inventory report with headers/footers |
| findTotalSales() | Calculates total extended prices |
| findAverageSales() | Calculates average sales per product |
| changeQuantity() | Search and update product quantity |
| changePrice() | Search and update product price |
| categoryReport() | Filter report by category |
| priceReport() | Filter products below max price |
| sortByNameReport() | Bubble sort by product name |
| sortByPriceReport() | Bubble sort by product price |
